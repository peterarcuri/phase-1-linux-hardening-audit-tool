# Phase 1 - Linux Hardening Audit Tool

A Bash-based Linux security auditing and hardening toolkit focused on:
- least privilege enforcement
- permission auditing
- firewall validation
- log management
- system security checks

---

## Features

- User & group auditing
- World-writable file detection
- SUID/SGID detection
- UFW firewall checks
- JSON reporting
